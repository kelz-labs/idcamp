# Point penting di materi jsx

- jsx hanya mengembalikan satu element

```jsx
// contoh yang benar 
<div>
  <p>Something</p>
</div> 

// salah, mesti dibungkus pakai div lagi atau react fragment(<></>)
<div>
  <p>Something 1</p>
</div>
<div>
  <p>Something 2</p>
</div>
```
- Walau mirip html, tapi jsx tidak bisa disamakan dengan html
